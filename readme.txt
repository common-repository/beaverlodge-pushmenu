== Description ==

**Requires Beaver Builder to work**

The Beaverlodge Pushmenu allows you to create a off-canvas menu using the simple drag and drop functionality of Beaver Builder.

You can choose between a dark or light off canvas, define the width and decide whether to have right or left menus.

Triggering the menu is easy with the menu button that can be text, icon or a combination of both.

All styling is easy at your fingertips without any need for knowing CSS or any other code.

### Features
* Easily choose which menu to use in you module
* Choose between left or right off canvas

== Frequently Asked Questions ==

= Can I use the plugin without Beaver Builder? =

No. Beaverlodge Pushmenu uses a custom Beaver Builder module to display the menu.

= Do I have to use the premium version of Beaver Builder? =

Yes, this version does not work with the Lite version of Beaver Builder.
